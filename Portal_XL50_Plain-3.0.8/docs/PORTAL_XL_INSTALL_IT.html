1. Installazione rapida

Se hai una conoscenza di base sull'uso FTP il server eseguirà phpBB3 Portal XL, quindi puoi possibile utilizzare questa procedura per iniziare rapidamente. Esegui questa proceduera solo se hai già installato una copia di phpbb 3.0.x!

  1. Decomprimi l'archivio Portal_XL50_Plain-3.0.7.rar in una directory locale del tuo computer.
  2. Scarica tutto il contenuto nella directory \root\ o nella \public_html\ del tuo server (rispettando la struttura).
  3. Modifica i permessi alle directories facendo in modo che siano scrivibili (777 o -rwxrwxrwx con il tuo client FTP):
    store/, cache/, files/, images/upload/avartars/, /config.php and /images/counter/ip.txt.
  4. Usa il tuo browser visitando la locazione dove hai posizionato phpBB3 Portal XL aggiungendo install/ o punta direttamente ad install/index.php, es. http://www.mydomain.com/phpBB3/install/index.php, http://www.mydomain.com/forum/install/index.php etc.
  5. Clicca sulla scheda INSTALLA e segui gli step successivi.
  6. phpBB3 Portal XL dovrebbe ora essere disponibile.

Se hai riscontrato problemi o non sai come procedere con una qualsiasi delle fasi di cui sopra sei pregato di leggere il resto di questo documento o vedi aggiornamento cache.

2. Requisiti

phpBB3 Portal XL necessita dei seguenti requisiti:

  • Un server web che supporti php, preferibilmente linux
  • Un sistema di database Sql
  • (MySQL 3.23 o oltre (MySQLi è supportato)
  • PHP 4.3.3+ (>=4.3.3, >4.4.x, >5.x.x, >6.0-dev (compatibile)) con supporto al database
  • getimagesize() - la funzione deve essere abilitata;
  • Le funzioni: zlib Compression support, Remote FTP support, XML support, Imagemagick support, GD Support migliorano il cms ma non sono indispensabili.

Se il tuo server di hosting non soddisfa i requisiti di cui sopra phpBB3 Portal XL non è per te.

3. Nuova installazione

Ora che hai scaricato uno dei pacchetti, puoi scegliere su come devi proseguire con l'installazione di Portal XL 5.0 Plain. Importante: Prima di fare qualunque cosa è indispensabile fare un backup totale del tuo database e dei files esistenti di phpBB3! Se non sei sicuro su come procedere potresti contattare il tuo hosting e chiedere un aiuto. ATTENZIONE! Se il tuo forum phpBB3 o Portal XL hanno avuto delle modifiche dovute all'installazione di mods diversi da quelli installati con Portal XL 5.0 ~ Plain, si perderanno tutte le modifiche!

3. Pacchetto Portal XL 5.0 Plain


Step da seguire:

- Scarica una recente copia di phpBB 3.0.x da phpbb.com o da phpbb.it se non è ancora installato. Se non è già installato procedi qui
- Scarica e decomprimi il paccheetto di phpBB 3.0.x su una directory del tuo computer
- Invia sul tuo server il contenuto della cartella phpBB3 nella directory root o public_html
- Posizionati sulla barra del tuo browser e scrivi ad esempio http://your-domain/install/, inizierà l'installazione e segui la guida in linea
- Elimina la cartella /install/ dopo l'installazione

Al prossimo step il tuo portale sarà stato creato.

- Scompatta l'archivio Portal_XL50_Plain-3.0.7.rar in una directory del tuo computer
- copia/invia tutto il contenuto (rispettando la struttura) del pacchetto nella directory \root\ o \public_html\ del tuo server.
- Posizionati sulla barra del tuo browser e scrivi ad esempio \install\index.php dopo il nome del tuo sito, lo script comincerà l'installazione dopo aver selezionato la scheda INSTALLA.

- Dopo l'installazione entra nel tuo ACP ed esegui gli step successivi:

- Tab Styles -> Style components -> Templates -> prosilver -> Aggiorna
- Tab Styles -> Style components -> Themes -> prosilver -> Aggiorna
- Tab Styles -> Style components -> Imagesets -> prosilver -> Aggiorna
- General Tab -> In main screen -> cancella la cache
- Aggiorna la cache del tuo browser

Pronto!

Sono un fun di Portal XL 5.0 ~ Plain ~ Il tuo insolito, folle portale per phpBB 3.0.x

Nota: se usi un'altra lingua oltre alla lingua inglese copia i file in \languages\xx\ dove xx corrisponde all'inziale (xx=it per l'italiano) della tua lingua.

4. Aggiornamento a phpBB3 Portal XL

Se stai utilizzando una versione stabile di phpBB3 Portal XL, l'aggiornamento a questa versione è molto semplice. Ora che hai scaricato uno dei pacchetti, puoi scegliere su come devi proseguire con l'installazione di Portal XL 5.0 Plain. Importante: Prima di fare qualunque cosa è indispensabile fare un backup totale del tuo database e dei files esistenti di phpBB3! Se non sei sicuro su come procedere potresti contattare il tuo hosting e chiedere un aiuto. ATTENZIONE! Se il tuo forum phpBB3 o Portal XL hanno avuto delle modifiche dovute all'installazione di mods diversi da quelli installati con Portal XL 5.0 ~ Plain, si perderanno tutte le modifiche!

4. Pacchetto Portal XL 5.0 Plain


Step da seguire:

- Scarica e decomprimi sul tuo computer una recente copia di phpBB 3.0.x da phpbb.com o da phpbb.it
- cancella il file config.php dalla directory /phpBB3/ del pacchetto
- Invia/sovrascrivi il tuo phpBB3 dalla root forum (il portale è perduto)
 - Scrivi sulla barra del browser http://your-domain/install/database_update.php e segui l'aggiornamento
- Elimina la cartalla /install

 

Ora il tuo portale è aggiornato.

ATTENZIONE!
Per aggiornare a questa versione correttamente è necessario eliminare tutti i file che non sono in uso più su Portal XL 5.0 ~ Plain o che sono stati spostati.
Non dimenticare la procedura descritta di seguito o verrai sorpreso da diversi errori.
- ELIMINA la directory /portal/* from your root
- ELIMINA in \language\en\portal.php, in \language\en\acp\portal_*.php, in \language\en\mods\average_statistics.php, in \adm\style\acp_portal*.html (tutti i files sono stati spostati).

- Decomprimi il file Portal_XL50_Plain-3.0.7.rar in una directory del tuo computer
- copia/invia tutto il contenuto (rispettando la struttura) del pacchetto nella directory \root\ o \public_html\.
- Scrivi sulla barra del tuo browser \install\index.php di seguito al nome del tuo sito e comincerà l'installazione potrai così scegliere la scheda aggiornamento.

- Dopo l'installazione entra nel tuo ACP ed esegui gli step successivi:

- Tab Styles -> Style components -> Templates -> prosilver -> Aggiorna
- Tab Styles -> Style components -> Themes -> prosilver -> Aggiorna
- Tab Styles -> Style components -> Imagesets -> prosilver -> Aggiorna
- General Tab -> In main screen -> cancella la cache
- Aggiorna la cache del tuo browser

Pronto!
