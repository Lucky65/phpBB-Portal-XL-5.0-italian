1. Der schnelle Einstieg

Wenn Du schon etwas Basiswissen (auch in FTP) mitbringst, und Du sicher bist, dass phpBB3 Portal XL auf deinem Server lauffähig ist, starte sogleich mit dem Schnelleinsteig.

  1. Entpacke das Portal_XL50_Plain-3.0.8.rar Archiv in einen Ordner auf deinem Rechner.
  2. In dem entpackten Ordner ist das Verzeichnis \root\. Den Inhalt dieses Verzeichnisses übertrage nun via FTP in das Root Verzeichnis deiner phpBB3 installation auf deinem Server. Beachte, das die Ordnerstrukturen exakt eingehalten werden und alle anwesenden Dateien überschrieben werden.
  3. Die Zugriffsrechte (CHMOD 777 or -rwxrwxrwx) musst Du für folgende Ordner/Dateien mit Hilfe deines FTP-Clienten setzen:
    store/, cache/, files/, images/upload/avartars/, /config.php und /images/counter/ip.txt.
  4. Nun begeben wir uns mit dem Browser Deiner Wahl zum Installationsort. Die Adresse sollte dann in etwa so aussehen: install/index.php, e.g. http://www.mydomain.com/phpBB3/install/index.php, http://www.mydomain.com/forum/install/index.php etc.
  5. Klicke nach erreichen der Seite oben auf den INSTALL tab, folge nun den angezeigten Schritten (Steps).
  6. phpBB3 Portal XL sollte nun bereit sein.

Sollte es wiedererwarten zu Problemen kommen, kann der Schritt: Zwischenspreicher erneuern Abhilfe schaffen.

2. Was ist zu Installation erforderlich?

phpBB3 Portal XL benötigt zur Installation einige wichtige Dinge, ohne die eine Installation nicht möglich ist.

  • Ein Webserver oder web hosting Konto oder ähnliches Betriebssystem (z.B. XAMPP unter WINDOWS) welches PHP unterstützt
  • Ein SQL DatenbankSystem:
    • MySQL 3.23 oder Höher (MySQLi unterstützt)
  • PHP 4.3.3+ (>=4.3.3, >4.4.x, >5.x.x, >6.0-dev (compatible)) mit unterstützung für Dein DatenbankSystem.
  • getimagesize() Function muss aktiviert sein.
  • Wenn zuvor genannte Anforderungen nicht gegeben sind, ist der Betrieb des phpBB3 Portal XL nicht möglich.
  • Folgende Optionen sind für den erfolgreichen sehr hilfreich, aber nicht unbedingt erforderlich.
    • zlib Compression Unterstützung
    • Remote FTP Unterstützung
    • XML Unterstützung
    • Imagemagick Unterstützung
    • GD Unterstützung

3. Neuinstallation

Es gibt zwei Pakete des Portals, Plain und Premod. Nachdem Du Dich nun für eines entschieden hast (Portal XL 5 Plain) und mit der Installation starten möchtest, solltest Du vorab folgende Schritte tun. WICHTIG: Vor der Installation empfehlen wir Dir ein komplettes Backup deiner jetzigen phpBB3 Datenbank sowie der gesamten phpBB3 Dateien die auf deinem Server liegen! Wenn Du unsicher bist, wie das richtig funktioniert, wende Dich an Deinen Provider, der sicherlich gerne eine Hilfestellung gibt. ACHTUNG! Sollte Dein phpBB3 bereits mit anderen Mods oder Veränderungen modifiziert sein, sind diese nach der Portal XL Installation verloren!

3. Portal XL 5.0 Plain Paket


Die Schritte zum phpBB3 + Portal XL:

- Besorge Dir die aktuelle phpBB3.0.1 Forumsversion auf phpbb.de (deutsch) oder phpbb.com (englisch) sofern dies noch nicht geschehen ist. Wenn schon installiert gehts hier weiter: Portal installalieren
- Enpacke phpBB 3.0.x in ein Verzeichnis auf Deinem Rechner
- Den Inhalt des Verzeichnisses in das Root des Servers laden
- Rufe die install auf z.B. http://your-domain/install/ und der Installer wird Dich durch die Installation führen
- Lösche oder benenne der Ordner /install/ um, wenn die Installation beendet ist.

Nun installieren wir das Portal.

- Entpacke das Archiv Portal_XL50_Plain-3.0.8.rar in ein Verzeichnis auf Deinem Rechner (falls noch nicht geschehen)
- Den gesamten Inhalt des Root-Ordner (/root/) lade nun in das Hauptverzeichnis (oder dein phpBB3 Verzeichnis) deines Servers hoch. Versichere Dich, das alles hochgeladen wurde und auch das die Ordnerstrukturen eingehalten wurden. Fragen nach überschreiben vorhandener Dateien ist mit JA zu beantworten.

ACHTUNG: Wenn keine Englische Installation vorgenommen wird kopiere aus dem entpackten Install-Ordner die \language\de\acp\portal_xl_install.php in das Verzeichnis (vom Rootverzeichnis ausgehend) mit Deiner Sprache. z.B. nach language\de\acp\. Vergisst Du diesen vorab-Schritt, wird es keine Installationstexte geben, da die benötigten Texte nicht gefunden werden können. Aus vorgegebenen Gründen ist dies nur so zu lösen!

- Rufe nun die Install im Browser auf: \install\index.php . Das Installationsscript wird geladen und Du klickst auf den TAB "Install".

- War die Installation Erfolgreich, so lösche den Ordner \install\ oder benenne ihn um, sofern dies nicht schon beim installieren automatisch passiert ist.

- Nach der Installation begib Dich ins ACP und mache folgendes:

- Tab Styles -> Style Komponenten -> Templates -> prosilver -> aktualisieren
- Tab Styles -> Style Komponenten -> Themes -> prosilver -> aktualisieren
- Tab Styles -> Style Komponenten -> Grafiksammlung -> prosilver -> aktualisieren
- Tab Allgemein-> In diesem Fenster ->Zwischenspeicher leeren
- Aktualisiere Deinen Browser und/oder leere dessen Zwischenspeicher ebenfalls.

Fertig!
