1. Snelle installatie

Indien U basiskennis van FTP gebruik hebt en deze dienst voor U beschikbaar is kunt U deze stappen volgen om snel met Portal XL aan de slag te kunnen.

  1. Pak het bestand Portal_XL50_Plain-3.0.8.rar uit naar een map op Uw systeem.
  2. Upload alle bestanden uit de map \root\ van het archieve (behoud de mapstructuur) naar een via het web toegankelijke map op uw server.
  3. Wijzig de toegangsrechten voor bestanden en mappen naar (777 of -rwxrwxrwx middels Uw FTP client):
    store/, cache/, files/, images/upload/avartars/, /config.php en /images/counter/ip.txt.
  4. Surf met Uw webbrowser naar de locatie waar de bestanden van het phpBB3 Portal XL te vinden zullen zijn met als aanvulling install/ of roep install/index.php direct op, zoals b.v.b. http://www.mydomain.com/phpBB3/install/index.php, http://www.mydomain.com/forum/install/index.php enz.
  5. Klik op de INSTALL ruiter, volg de stappen die verschijnen.
  6. phpBB3 Portal XL zal hierna beschikbaar zijn voor gebruik.

Indien U problemen ervaart of niet weet hoe door te gaan met de opgesomde stappen lees alstublieft het overige van dit document of zie verversen cache.

2. Vereisten

phpBB3 Portal XL heeft enkele vereisten die moeten worden vervult alvorens U kunt doorgaan met het installeren en gebruiken.

  • Een webserver of web hosting account met ondersteuning voor PHP
  • Een SQL database systeem:
    • MySQL 3.23 of hoger (MySQLi ondersteunt)
  • PHP 4.3.3+ (>=4.3.3, >4.4.x, >5.x.x, >6.0-dev (compatibel)) met ondersteuning voor de databank die U van plant bent te gebruiken.
  • getimagesize() functie dient te zijn geactiveerd.
  • Deze optionele functies of modules binne PHP zullen toegevoegde opties mogelijk maken, ze zijn echter niet verplicht.
    • zlib Compressie ondersteuning
    • Remote FTP ondersteuning
    • XML ondersteuning
    • Imagemagick ondersteuning
    • GD ondersteuning

Indien Uw server of account niet aan deze minimale vereisten voldoet zijn wij bang dat phpbb3 Portal XL niet voor u geschikt is.

3. Nieuwe installatie

U zult een van twee beschikbare portaal pakketten gedownload hebben en uw keuze stelt vast wat u moet doen, in dit geval Portal XL 5.0 Plain. Merk op: wij raden ten zeerste aan dat U een volledige back-up van Uw databank en aanwezige phpbb3 bestanden maakt alvorens door te gaan! Indien u onzeker bent hoe dit te bereiken vraag Uw systeembeheerder om advies. ATTENTIE! Mocht Uw bestaande phpBB3 of Portal XL set-up gewijzigd zijn door modificaties anders dan diegene die Portal XL 5.0 ~ Plain gebruikt, zullen deze door installatie van het portaal verloren gaan!

Merk op: gebruikt U een andere taal dan Engels is het aan te bevelen om het bestand \language\en\acp\portal_xl_install.php naar de corresponderende map van Uw geïnstalleerde taal te kopiëren om de installatie tekst (in Engels) beschikbaar te hebben mits er onder \languages\xx\ geen corresponderend bestand aanwezig mocht zijn. Dit om foutmeldingen tijdens een meertalige configuratie/installatie te voorkomen.

3. Portal XL 5.0 Plain


Benodigde stappen:

- Download Uw eigen kopie phpBB 3.0.x van phpbb.com mits nog niet gedaan. Mits geïnstallerd ga hier verder
- Pak phpBB 3.0.x uit naar een tijdelijke map
- Upload de map phpBB3 naar Uw forum root
- Browse naar b.v.b. http://your-domain/install/ en het installatie script zal opstarten
- Verwijder de map /install/ als de installatie af is gesloten

De volgende stappen zullen het portaal installeren.

- Pak archief Portal_XL50_Plain-3.0.8.rar uit naar een tijdelijke map
- Upload alle bestanden van het archief (behoud de mapstructuur) uit de map \root\ naar Uw forum root en vergewis U dat alles op zijn plaats is.
- Browse naar \install\index.php, klik op de INSTALL ruiter, volg de stappen die verschijnen.

- Na installatie ga naar het ACP en voer de stappen hieronder uit:

- Tab Styles -> Style components -> Templates -> prosilver -> Verversen
- Tab Styles -> Style components -> Themes -> prosilver -> Verversen
- Tab Styles -> Style components -> Imagesets -> prosilver -> Verversen
- General Tab -> In main screen -> Purge the cache
- Ververs Uw browsers cache

Klaar!

4. Portal XL 5.0 Plain


Benodigde stappen:

Enkel nodig als er nog geen recente phpBB 3.0.x in gebruik is. Mits geïnstallerd ga hier verder.

- Download Uw eigen kopie phpBB 3.0.x van phpbb.com mits nog niet gedaan.
- Pak phpBB 3.0.x uit naar een tijdelijke map
- Verwijder bestand config.php uit de map /phpBB3/ van het paket
- Upload/vervang de map phpBB3 naar Uw forum root (portaal is nu verwijderd)
- Browse naar b.v.b. http://your-domain/install/database_update.php en het update script zal opstarten
- Verwijder de map /install/ als de installatie af is gesloten

De volgende stappen zullen het portaal installeren.

ATTENTIE!
To update to this version properly some essential steps are needed to follow up at your running site.
These steps are neccessary to delete ALL files which are not in use anymore from on Portal XL 5.0 ~ Plain or which where moved.
Don't forget the steps below or you will be surprised by a mess if you'r using the block management at later stage (wrong blocks, dubble blocks, wrong block layout etc.).
- REMOVE directory /portal/* from your root
- REMOVE for this release in \language\en\portal.php, in \language\en\acp\portal_*.php, in \language\en\mods\average_statistics.php, in \adm\style\acp_portal*.html (all those files where moved into a different place).

- unpack archive Portal_XL50_Plain-3.0.8.rar to a temporary directory
- copy/uload all content (retaining the directory structure) from the package directory \root\ to your forum root and be sure all is in place.
- browse to \install\index.php to start the installation script and choose tab "Update" (no database changes for Plain 0.2).

- Na installatie ga naar het ACP en voer de stappen hieronder uit:

- Tab Styles -> Style components -> Templates -> prosilver -> Verversen
- Tab Styles -> Style components -> Themes -> prosilver -> Verversen
- Tab Styles -> Style components -> Imagesets -> prosilver -> Verversen
- General Tab -> In main screen -> Purge the cache
- Ververs Uw browsers cache

Ready!
