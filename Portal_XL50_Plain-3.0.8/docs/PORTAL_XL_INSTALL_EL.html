1. Quick install

If you have basic knowledge of using FTP and are sure your hosting service or server will run phpBB3 Portal XL you can use these steps to quickly get started.

  1. Decompress the Portal_XL50_Plain-3.0.8.rar archive to a local directory on your system.
  2. Upload all the files contained in direcory \phpbb3\ of this archive (retaining the directory structure) to a web accessible directory on your server or hosting account.
  3. Change the permissions on the following directories to be writable by all (777 or -rwxrwxrwx within your FTP Client):
    store/, cache/, files/, images/upload/avartars/, /config.php and /images/counter/ip.txt.
  4. Using your web browser visit the location you placed phpBB3 Portal XL with the addition of install/ or pointing directly to install/index.php, e.g. http://www.mydomain.com/phpBB3/install/index.php, http://www.mydomain.com/forum/install/index.php etc.
  5. Click the INSTALL tab, follow the steps as requested.
  6. phpBB3 Portal XL should now be available.

If you experienced problems or do not know how to proceed with any of the steps above please read the rest of this document or see refresh cache.

2. Requirements

phpBB3 Portal XL has a few requirements which must be met before you are able to install and use it.

  • A webserver or web hosting account running on any major Operating System with support for PHP
  • A SQL database system:
    • MySQL 3.23 or above (MySQLi supported)
  • PHP 4.3.3+ (>=4.3.3, >4.4.x, >5.x.x, >6.0-dev (compatible)) with support for the database you intend to use.
  • getimagesize() function need to be enabled.
  • These optional presence of the following modules within PHP will provide access to additional features, but they are not required.
    • zlib Compression support
    • Remote FTP support
    • XML support
    • Imagemagick support
    • GD Support

If your server or hosting account does not meet the requirements above we are afraid phpBB3 Portal XL is not for you.

3. New installation

You would have downloaded one of two packages and your choice determines what you need to do, in this case Portal XL 5.0 Plain. Please Note: That before updating we heavily recommend you do a full backup of your database and existing phpBB3 files! If you are unsure how to achieve this please ask your hosting provider for advice. ATTENTION! If your former phpBB3 or Portal XL setup was modificated by mods other than those who come with Portal XL 5.0 ~ Plain, you will loose all those changes!

3. Portal XL 5.0 Plain package

Note: before proceding and if you are using any other language than English copy file \language\en\acp\portal_xl_install.php to the corresponding directory of your installed language to have the text (in English) available if under \languages\xx\ no corresponding file is available.

Steps to do:

- Get your own recent copy of phpBB 3.0.x from phpbb.com if not installed already. If installed proceed here
- Unpack phpBB 3.0.x to a temporary directory
- Upload folder phpBB3 to your forum root
- Browse to eg. http://your-domain/install/ and the installer will activate
- Remove folder /install/ when installation is ready

The next steps will creating your portal.

- unpack archive Portal_XL50_Plain-3.0.8.rar to a temporary directory
- copy/uload/overwrite all content (retaining the directory structure) from the package directory \phpbb3\ to your forum root eg. \public_html\ and be sure all is in place.
- browse to \install\index.php to start the installation script and choose tab "Install".

- After installation enter the ACP and do all the steps below:

- Tab Styles -> Style components -> Templates -> prosilver -> Refresh
- Tab Styles -> Style components -> Themes -> prosilver -> Refresh
- Tab Styles -> Style components -> Imagesets -> prosilver -> Refresh
- General Tab -> In main screen -> Purge the cache
- Refresh your browsers cache

Ready!

PS: visit our NEW style Demo/Download for Portal XL 5.0 ~ Plain aware styles

4. Updating a former release of phpBB3 Portal XL

If you are currently using a stable release of phpBB3 Portal XL updating to this version is straightforward. You would have downloaded one of two packages and your choice determines what you need to do, in this case Portal XL 5.0 Plain. Please Note: That before updating we heavily recommend you do a full backup of your database and existing phpBB3 Portal XL files! If you are unsure how to achieve this please ask your hosting provider for advice. ATTENTION! If your former phpBB3 or Portal XL setup was modificated by mods other than those who come with Portal XL 5.0 ~ Plain, you will loose all those changes!

4. Portal XL 5.0 Plain package

Note: before proceding and if you are using any other language than English copy file \language\en\acp\portal_xl_install.php to the corresponding directory of your installed language to have the text (in English) available if under \languages\xx\ no corresponding file is available.

Steps to do:

Only needed if not already using a recent phpBB 3.0.x. If installed proceed here

- Get your own recent copy of phpBB 3.0.x from here http://www.phpbb.com/downloads/
- Unpack phpBB 3.0.x to a temporary directory
- Delete file config.php from folder /phpBB3/ of that package
- Upload/overwrite folder phpBB3 to your forum root (portal is lost)
- Browse to eg. http://your-domain/install/database_update.php and run the updater
- Remove folder /install/

The next steps will re-creating your portal.

ATTENTION!
To update to this version properly some essential steps are needed to follow up at your running site.
These steps are neccessary to delete ALL files which are not in use anymore from on Portal XL 5.0 ~ Plain or which where moved.
Don't forget the steps below or you will be surprised by a mess if you'r using the block management at later stage (wrong blocks, dubble blocks, wrong block layout etc.).
- REMOVE directory /portal/* from your root
- REMOVE for this release in \language\en\portal.php, in \language\en\acp\portal_*.php, in \language\en\mods\average_statistics.php, in \adm\style\acp_portal*.html (all those files where moved into a different place).

- unpack archive Portal_XL50_Plain-3.0.8.rar to a temporary directory
- copy/uload/overwrite all content (retaining the directory structure) from the package directory \phpbb3\ to your forum root eg. \public_html\ and be sure all is in place.
- browse to \install\index.php to start the installation script and choose tab "Update" (no database changes for Plain 0.2).

- After installtion enter the ACP and do all the steps below:

- Tab Styles -> Style components -> Templates -> prosilver -> Refresh
- Tab Styles -> Style components -> Themes -> prosilver -> Refresh
- Tab Styles -> Style components -> Imagesets -> prosilver -> Refresh
- General Tab -> In main screen -> Purge the cache
- Refresh your browsers cache

Ready!
