Portal_XL50_Premod-3.0.8.rar

This archive contains Portal XL 5.0 Premod 0.3 along phpBB 3.0.8.

However the package is tested, no guaranty is given on proper working.

How to install and/or update Portal XL 5.0 Premod 0.3 is decribed clearly 
in directory \phpbb3\docs\PORTAL_XL_INSTALL.html of this archive.


Have fun with another fine release of Portal XL 5.0 Premod!

Attention! If your phpBB 3.0.8 is changed by custom modifications don't 
use this package as you will loose all the changes made.

DROP TABLE IF EXISTS ajax_chat_online;
CREATE TABLE ajax_chat_online (
	userID INT(11) NOT NULL,
	userName VARCHAR(64) NOT NULL,
	userRole INT(1) NOT NULL,
	channel INT(11) NOT NULL,
	dateTime DATETIME NOT NULL,
	ip VARBINARY(16) NOT NULL
) DEFAULT CHARSET=utf8 COLLATE=utf8_bin;

DROP TABLE IF EXISTS ajax_chat_messages;
CREATE TABLE ajax_chat_messages (
	id INT(11) NOT NULL AUTO_INCREMENT,
	userID INT(11) NOT NULL,
	userName VARCHAR(64) NOT NULL,
	userRole INT(1) NOT NULL,
	channel INT(11) NOT NULL,
	dateTime DATETIME NOT NULL,
	ip VARBINARY(16) NOT NULL,
	text TEXT,
	PRIMARY KEY (id)
) DEFAULT CHARSET=utf8 COLLATE=utf8_bin;

DROP TABLE IF EXISTS ajax_chat_bans;
CREATE TABLE ajax_chat_bans (
	userID INT(11) NOT NULL,
	userName VARCHAR(64) NOT NULL,
	dateTime DATETIME NOT NULL,
	ip VARBINARY(16) NOT NULL
) DEFAULT CHARSET=utf8 COLLATE=utf8_bin;

DROP TABLE IF EXISTS ajax_chat_invitations;
CREATE TABLE ajax_chat_invitations (
	userID INT(11) NOT NULL,
	channel INT(11) NOT NULL,
	dateTime DATETIME NOT NULL
) DEFAULT CHARSET=utf8 COLLATE=utf8_bin;

## Welcome PM Settings
Enter ACP -> SYSTEM -> Module management -> Administration Controle panel -> enter the .MODS tab ->
at bottom in field "Create New Module" enter ACP_WELCOME_PM and create the new module and enable, in drop down "Add Module"
search module "Welcome PM Settings" and add this module and enable. Proceed to .MODS at the bottom the new WPM will
be visible to configure from.

## time zone
Go to your Board Settings and choose an appropriate time zone or approve the displayed one by hitting "Submit" if it is already correct.
Run convert_timezones.php to convert existing user time zones to the new city-based format.

## Country Flags
After you finished above step, you need install 3 modules of this MOD in your ACP manually.
This MOD have all 3 module: 2 active and 1 hidden.
Clear all files in "cache" directory without .htaccess file.
Go to ACP -> SYSTEM (tab) -> Module management -> Administration Control Panel.

Install module 1 & 2:
Click: Users and Groups -> Users, look at the end of page, select from dropdown list (first list beside "Add module" button):
+ Category: Country flags [acp_flags], Name: Manage country flags. (This is the active module)
+ Category: User management [acp_users], Name: Country flag. (This is hidden module, you will see "Country flag [Hidden module]" text after enable this module)

Install module 3:
Click: General -> Board configuration, look at the end of page, select from dropdown list (first list beside "Add module" button):
+ Category: Board management [acp_board], Name: Country flag settings. (This is the active module)
