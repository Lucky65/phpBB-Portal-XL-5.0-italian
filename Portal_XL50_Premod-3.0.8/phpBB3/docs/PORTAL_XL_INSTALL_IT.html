1. Installazione rapida

Se disponi di una conoscenza di base ad usare un client FTP è possibile utilizzare questa procedura per essere subito operativi.

  1. Decomprimi l'archivio Portal_XL50_Premod-3.0.8.rar in una directory temporanea del tuo computer.
  2. Carica tutti i file contenuti nella directory \phpBB3\ dell'archivio Portal_XL50_Premod-3.0.8.rar (mantenendo la struttura delle directory) in una directory web accessibile sul tuo server o account di hosting.
  3. Modifica le autorizzazioni (chmods) per le seguenti directory affichè siano scrivibilii ( 777-rwxrwxrwx tramite il tuo client FTP): / cache, / store, / file, / images / avatars / upload, / dl_mod / thumbs e tutte le loro sottocartelle, / dl_mod / downloads e tutte le loro sottocartelle, / gallery / images e tutte le loro sottocartelle, / phpbb_seo / cache, / gym_sitemaps / cache, / mods / toplist_mod / cache, / mods / toplist_mod / images, / Arcade, / arcade / gamedata, / arcade / games, / arcade / install e almeno / images / counter / ip.txt.
  4. Controlla se il tuo config.php è scrivibile da tutti (777-755).
  5. Scrivi sulla barra del tuo browser install_portal / preceduto dall'indirizzo del tuo sito ad esempio tuosito.it/install_portal/ index.php, oppure se il tuo forum phpbb 3.0.8 si trova in una sottodirectory ad esempio \forum\  scrivi ad esempio http://www.mydomain.com/forum/install_portal/index.php ecc
  6. Clicca sulla scheda "Installazione base" per installare la versione base (Portal XL 5.0 Plain) e segui le istruzioni fornite. Non eseguire il login, ricorda che hai i files della Premod!
  7. Clicca sulla scheda "Aggiornamento a Premod", poi su "Aggiornamento del database", Se l'aggiornamento è completato, puoi eseguire il login per accedere alla tua Portal XL 5.0 Premod.
  8. Loggati come amministratore (devi essere un founder) ed esegui sul tuo browser /convert_timezones.php, ad esempio http://tuosito/phpBB3/convert_timezones.php per convertire tutti gli utenti al nuovo formato del fuso orario. Elimina questo file quando hai finito!
  9. phpBB3 Portal XL dovrebbe ora essere disponibile.

Ora è il momento di modificare i permessi per tutti i mods. Loggati nel tuo ACP, vai nella scheda permessi, in Ruoli Admin / Ruoli Utenti / Ruoli Moderatori / Ruoli Forum... Definisci ora i permessi per i ruoli.

Se hai avuto problemi o non sai come procedere con una qualsiasi delle fasi di cui sopra, leggi il resto di questo documento o vedi aggiornamento della cache.

2. Requisiti

phpBB3 Portal XL è dotato di alcuni requisiti che devono essere soddisfatti affinchè tu sia in grado di installare e utilizzare.

  • Un server web o un account web hosting in esecuzione su qualsiasi sistema operativo principale, con supporto per PHP
  • Un sistema di database SQL:
    • MySQL 3.23 o superiore (MySQLi supportato)
  • PHP 4.3.3 + (> = 4.3.3,> 4.4.x,> 5.xx,> 6.0-dev (compatibile)) con il supporto per il database che si intende utilizzare.
  • getimagesize () la funzione deve essere attivata.
  • Questo requisito è facoltativo e fornirà l'accesso a funzioni aggiuntive.
    • Compressione zlib
    • FTP remoto
    • Supporto XML
    • Supporo GD
    • Supporto Exif
    • Supporto Curl

Se il tuo server o account di hosting non soddisfa i requisiti di cui sopra abbiamo paura phpBB3 Portal XL non è per voi.

5. Eliminare Portal XL 5.0

Per eliminare Portal XL 5,0 Premod completamente senza lasciare alcun residuo:

  • Avvia lo script di installazione e scegliere sulla scheda "ELIMINA PORTALE".

    Questa azione rimuoverà tutti i dati dal database, con l'eliminazione del portale il tuo forum diventerà come una versione normale di phpBB. L'eliminazione del portale non rimuove i mods di terze parti, se vuoi eliminarli devi utilizzare lo script di installazione di ogni mod.

    Il prossimo passo sarà quello di eliminare tutti i file all'interno del tuo forum directory principale, tranne il file config.php (questo file è molto importante per la connessione al database). Se vuoi conservare gli allegati non eliminare la directory /files/, lo stesso per le icone, in tal caso non eliminare la cartella /images/, se vuoi conservare avatars, icone, ranks., ecc. Scarica una versione recente di phpBB 3.0.x, elimina il file il file config.php e la cartella \install\ dal pacchetto e sovrascivi l'intero contenuto sul tuo sito.
