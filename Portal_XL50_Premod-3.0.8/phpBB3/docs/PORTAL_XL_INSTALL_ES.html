1. Instalación rápida

Si usted tiene conocimiento básico del uso de FTP y está seguro de su servicio de hosting o servidor se ejecutará phpBB3 Portal XL puede utilizar estos pasos para comenzar a trabajar rápidamente.

  1. Tener una phpBB 3.0.8 instalado, o un 3.0.7 actualizado a 3.0.8 ya (lo puede conseguir en @ phpbb.com)!
  2. Descomprimir el Portal_XL50_Premod-3.0.8.rar archivo a un directorio local en el sistema.
  3. Cargar todos los archivos contenidos en el directorio \phpbb3\ de este archivo (conservar la estructura de directorios) para una web directorio accesible en su servidor o cuenta de alojamiento.
  4. Compruebe si el archivo config.php esta en su foro directorio principal si es capaz de escribir todos los (777/755).
  5. Cambiar los permisos en los siguientes directorios (777/755 o -rwxrwxrwx dentro de su cliente FTP): /cache, /store, /files, /images/avatars/upload, /dl_mod/thumbs y todas sus subcarpetas, /dl_mod/downloads y todas sus subcarpetas, /gallery/images y todas sus subcarpetas, /phpbb_seo/cache, /gym_sitemaps/cache, /mods/toplist_mod/cache, /mods/toplist_mod/images, /arcade, /arcade/gamedata, /arcade/games, /arcade/install y por lo menos /images/counter/ip.txt.
  6. Usando su navegador web visite el lugar que coloco phpBB3 Portal XL con la adición de install_portal/ o navegar directamente a install_portal/index.php, por ejemplo http://www.mydomain.com/phpBB3/install_portal/index.php etc.
  7. Haga clic en la pestaña "Install Basic" para instalar el portal básico primero siguiendo los pasos determinados. no se conecte a través del portal antes de que se ha hecho todos los pasos! La secuencia de comandos de instalación le guiará a través de todos los pasos necesarios.
  8. Vaya a /convert_timezones.php, por ejemplo http://www.mydomain.com/phpBB3/convert_timezones.php para convertir todos los usuarios zonas horarias en el nuevo formato. Quitar ese archivo cuando haya terminado!
  9. phpBB3 portal XL ahora debería estar disponible.

Si experimentado problemas o no saben cómo proceder con cualquiera de los pasos anteriores por favor, lea el resto de este documento o consulte actualizar caché.

2. Requisitos

phpBB3 portal XL tiene unos requisitos que deben cumplirse antes de poder instalar y utilizarlo.

  • Un servidor web o cuenta de hospedaje web se ejecuta en cualquier sistema operativo con soporte para PHPUn sistema de base de datos SQL:
    • MySQL 3.23 o superior (MySQLi suportado)
  • PHP 4.3.3+ (>=4.3.3, >4.4.x, >5.x.x, >6.0-dev (compatible)) con soporte de la base de datos que pretende utilizar.
  • getimagesize() necesidad de que se habilite la función.
  • Opcionalmente, la presencia de los siguientes módulos en PHP proporcionan acceso a funciones adicionales.
    • zlib Compresión ssoporte
    • Remoto FTP soporte
    • XML soporte
    • GD Ssoporte
    • Exif Ssoporte
    • Curl Ssoporte

Si su servidor o cuenta de hospedaje no cumple los requisitos anteriores nos tememos phpBB3 Portal XL no es para ti.

5. Remover Portal XL 5.0

Para remover Portal XL 5.0 Premod completamente sin dejar ningún residuo:

  • Inicie el script de instalación y seleccione la pestaña  "REMOVER PORTAL".

    Esta acción eliminará todas Portal XL Premod relacionadas y descargar registros de base de datos mod incluidos los cambios realizados a phpBB3 en tablas predeterminadas, tales como módulos, grupos, BBCodes personalizados, usuarios, etc Después de este procedimiento usted tendrá un claro phpBB 3.0.x base de datos con tablas de base de datos y las entradas de blog del usuario, fotos y Arcade (esta base de datos de las entradas nunca son tocados por el portal de la secuencia de comandos de instalación). Para quitar las tablas usar los scripts de instalación de estos mods por favor.

    El próximo paso será eliminar todos los archivos en el directorio principal de su foro, EXCEPTO archivo config.php (este archivo es muy importante para conectar a su base de datos). Si tiene archivos adjuntos dejar directorio /files/ intacto. Si usted tiene otros extras colocado como iconos, sonrisas, avatares, imágenes comprobar en este rango, etc. Tienen un nuevo phpBB 3.0.x distribución, por un lado, eliminar el archivo config.php y carpeta \install\ de ese paquete. Cargar el paquete fresco a su espacio de foro, listos.
